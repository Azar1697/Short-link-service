ShortLink Service — это приложение для генерации коротких ссылок с использованием реального API [clck.ru](https://clck.ru). Программа поддерживает управление короткими ссылками, включая их редактирование, удаление и просмотр.

## Как-же пользоваться сервисом?

1. Скачайте или клонируйте репозиторий данной командой:
   ```bash
   git clone https://github.com/your-repo/shortlink-service.git

3. Скомпилируйте код. Выполните команду для компиляции:
   ```bash
   javac -d out src/*.java

4. Запустите приложение. Перейдите в папку out и выполните:
   ```bash
   java ShortLinkService

Какие-же команды поддерживает??

1. Зарегистрироваться
  Описание: Регистрация нового пользователя.
  Как использовать:
    Введите число 1.
    Укажите имя пользователя.
    Получите уникальный UUID.

2. Сократить ссылку
  Описание: Генерация короткой ссылки.
  Как использовать:
    Введите число 2.
    Укажите ваш UUID, сгенерированный при регистрации.
    Введите длинный URL для сокращения.
    При необходимости задайте максимальное количество переходов или используйте по умолчанию (указано в config.properties).

3. Отредактировать
  Описание: Редактирование параметров короткой ссылки.
  Как использовать:
    Введите число 3.
    Укажите ваш UUID.
    Введите короткую ссылку, которую хотите изменить.
    Задайте новое максимальное количество переходов и срок действия ссылки в часах.

4. Удалить
    Описание: Удаление короткой ссылки.
    Как использовать:
      Введите число 4.
      Укажите ваш UUID.
      Укажите короткую ссылку для удаления.

5. Посмотреть ссылку
    Описание: Перейти по короткой ссылке.
    Как использовать:
      Введите число 5.
      Укажите короткую ссылку. Программа откроет связанный длинный URL в браузере.

6. Выход
    Описание: Завершение работы программы.
    Как использовать:
      Введите число 6.

Как протестировать код????

Настройте среду разработки.

Откройте проект в вашей IDE (например, IntelliJ IDEA, VS Code).
Убедитесь, что установлена JDK версии 8 или выше.
Проверьте подключение к интернету. API clck.ru требует подключения для работы.

Создайте тестовые данные.

Зарегистрируйте нового пользователя.
Создайте несколько коротких ссылок.
Протестируйте команды.

Проверьте команды shorten, edit, delete, browse.
Убедитесь, что ссылки корректно работают, редактируются и удаляются.
Запустите программу в терминале.

Выполните команды из раздела "Как пользоваться сервисом".
Отладка.

Если что-то пошло не так, проверьте вывод ошибок в консоли.
Убедитесь, что URL передаётся корректно (например, начинается с http:// или https://).
